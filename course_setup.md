# Course Setup

At the begining of the course you will be provided with a zip "student scaffolding" which contains a base structure of a project that serves as a starting point and helpful claude skills that will assist you through the assignments.

Each week you will be provided a notebook and adaptation prompt that will be required to complete your assignments.

# Setup steps
1. Unzip the "student scaffolding" at location of your choice locally

## Colab
- Upload the "student scaffolding" to your google drive, this will be needed to setup your environment within colab.


## Local
Fire up claude in the "student scaffolding folder", if you don't have it set it up https://code.claude.com/docs/en/quickstart 

Once you are in claude, ask it to "setup the repository" this will trigger the skill that will run through the setup steps to make sure you have the following
    - UV https://docs.astral.sh/uv/getting-started/installation/ (python package manager)
    - Ollama https://ollama.com/ (Inference engine)


# Reciving Notebooks and Adaptation Prompts

Each week when you recieve the notebook you need to do two things,
    - copy the notebook locally in the notebooks folder
    - Upload the notebook to drive in the notebooks folder inside the "student scaffolding"
